## E-commerce Sales Analysis

This project analyzes e-commerce sales data to summarize performance and identify basic trends
that support business understanding and data-driven decision-making.

### Overview
- Performed data cleaning and exploratory analysis on anonymized sales data.
- Generated summary statistics and insights on monthly sales performance.

### Tools & Skills
- Python (Pandas)
- Data cleaning and aggregation
- Exploratory data analysis (EDA)

### Files
- Notebook: Step1_Data_Summary.ipynb  
- Dataset: data/sales_march_anonymized.csv  

*Note: This project is an early-stage analysis focusing on data preparation and summary reporting.*
